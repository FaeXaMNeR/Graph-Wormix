# Worms Math Revolution

Сетевая артиллерийская игра на Python + pygame. Игроки стреляют по друг другу траекториями, заданными математическими функциями `y = f(x)`.

## Установка

```
pip install pygame
```

Сервер (`server.py`) требует только стандартную библиотеку Python.

## Запуск

1. На одном компьютере в локальной сети запустите сервер:
```
python server.py --port 5555 --players 2
```
`--players N` — игра начнётся автоматически, когда подключится N игроков (от 2 до 6).

2. Каждый игрок запускает клиент, указав IP-адрес сервера:
```
python client.py 192.168.1.50 5555 Имя
```

## Управление

- **Кликните в текстовое поле** внизу экрана и введите функцию `y = f(x)`
- **Enter** или кнопка **"Огонь!"** — выстрел (только в свой ход)
- **Кнопка ←/→** —.flip оси X (стрелка меняет направление)
- **`move dx dy`** — перемещение (например `move 1 -2`), `dx² + dy² < 8`
- **Координаты курсора** в локальной СК отображаются в правом нижнем углу

### Разрешённые функции
`sin cos tan asin acos atan sinh cosh tanh exp log log10 sqrt abs floor ceil pow`

Константы: `pi`, `e`. Переменная: `x`.

Примеры: `sin(x)/4`, `x*0.3`, `x**2/500 - 60`, `-5*x`

## Правила

- Поле **1600×900** с процедурными препятствиями (квадраты, круги, прямоугольники, 30% заполнение).
- У каждого игрока **5 HP** и **своя система координат**, повёрнутая на случайный угол.
- Углы обновляются каждый новый раунд (когда все игроки сделали ход).
- В свой ход вводится функция `y = f(x)` в **локальной** (повёрнутой) системе координат — это траектория снаряда.
- Шум траектории растёт линейно с расстоянием, отклонения **перпендикулярны касательной** траектории.
- Снаряд разрушает препятствия при попадании (круглая область взрыва).
- Игра заканчивается, когда остаётся 0 или 1 игрок в живых.
- По окончании — голосование за реванш.

### Спавн
- Каждый игрок выбирает стартовую позицию кликом по полю.
- Если позиция на препятствии или слишком близко к другому — **все** расставляются заново.
- После 3 collective неудач — авто-расстановка.

## Бонусы

Появляются на чётных раундах начиная со 2-го. Один бонус на карте, один тип не повторяется 2+ раз подряд.

| Бонус | Эффект |
|-------|--------|
| **damage** (DMG) | Следующий выстрел наносит 2 HP (сохраняется при ходе) |
| **shield** (SHD) | Блокирует одно попадание до следующего хода |
| **double_shot** (x2) | Два выстрела подряд в одном ходу |
| **angle_reset** (ANG) | В следующем ходу оси ставятся в положение 0° |

Подбираются **выстрелом** (попадание в бонус) или **движением** (проход через бонус).

Отображаются как пульсирующий фиолетовый квадрат 16×16 px. У игрока с бонусом — индикатор рядом с HP-баром.
