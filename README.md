# Сетевая артиллерийская игра (Worms-like), Python + pygame

## Установка
```
pip install pygame
```
Сервер требует только стандартную библиотеку Python (без зависимостей).

## Запуск

1. На одном компьютере в локальной сети запустите сервер:
```
python3 server.py --port 5555 --players 2
```
`--players N` — игра автоматически начнётся, когда подключится N игроков.

2. Каждый игрок запускает клиента, указав IP-адрес сервера в локальной сети:
```
python3 client.py 192.168.1.50 5555 Vasya
```

## Правила
- Поле 1280×720, на нём несколько препятствий (квадраты и круги).
- У каждого игрока — кружок с 10 HP и СВОЯ система координат,
  повёрнутая на случайный угол; угол обновляется каждый новый раунд (когда все сделали ход).
- В свой ход введите функцию `y = f(x)` в локальной (повёрнутой) системе координат —
  это и есть траектория снаряда. Разрешены: `sin cos tan asin acos atan sinh cosh tanh
  exp log log10 sqrt abs floor ceil pow`, константы `pi e`, переменная `x`.
  Примеры: `sin(x)/4`, `x*0.3`, `x**2/500 - 60`.
- Если снаряд попадает в игрока — тот теряет 1 HP (можно попасть и в себя!).
- Если снаряд попадает в препятствие — оно частично разрушается («взрыв» вырезает
  круглую область вокруг точки попадания).
- На своём ходу на поле рисуются ваши оси координат (синяя — X, оранжевая — Y) для прицеливания.
- Игра заканчивается, когда остаётся 0 или 1 игрок в живых.

## Архитектура
- `shared.py` — константы и безопасный парсер выражений (через `ast`, без `eval` произвольного кода).
- `server.py` — авторитетный сервер: хранит состояние, генерирует препятствия,
  считает порядок ходов, симулирует траекторию и взрывы, транслирует JSON-сообщения по TCP.
- `client.py` — pygame-клиент: рисует поле/игроков/оси, поле ввода функции, анимацию выстрела.

Протокол — построчный JSON по TCP (newline-delimited), сетка препятствий передаётся
как base64 от компактного байтового массива.
